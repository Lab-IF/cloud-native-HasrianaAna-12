## 📝 Kuis Singkat

### 1. Next.js adalah...

**Jawaban:** _(Next.js adalah framework yang membungkus React. Dibuat oleh Vercel. Dia menambahkan semua yang React tidak punya)_

### 2. Mana yang TIDAK perlu di-install manual di Next.js?

**Jawaban:** _(React dan React-DOM. Saat kamu melakukan instalasi Next.js (menggunakan npx create-next-app), paket-paket utama seperti React sudah otomatis terpasang di dalamnya.)_

### 3. Saat install Next.js, agar menggunakan JavaScript (bukan TypeScript), kita harus pilih...

**Jawaban:** _(Pilih opsi "No" ketika muncul pertanyaan "Would you like to use TypeScript?" pada command prompt saat proses instalasi.)_

### 4. Apakah komponen React bisa dipakai di Next.js?

**Jawaban:** _(Ya, tentu saja. Next.js dibangun di atas React, sehingga semua komponen standar React bisa digunakan secara langsung di dalam proyek Next.js.)_

### 5. Sebutkan minimal 3 fitur yang Next.js berikan di atas React biasa!

**Jawaban:** _(1.  File-based Routing: Membuat rute halaman otomatis hanya dengan menambahkan file di dalam folder app atau pages.
2.  Server-Side Rendering (SSR): Kemampuan untuk merender halaman di sisi server sehingga lebih cepat dibaca oleh mesin pencari (SEO friendly).
3.  Built-in Optimization: Optimasi otomatis untuk Gambar (next/image), Font (next/font), dan Script agar performa website lebih maksimal.)_
